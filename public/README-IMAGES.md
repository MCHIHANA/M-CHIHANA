# Project Images Guide

To display your project images, add the following image files to the `public` folder:

## Required Images:

1. **farmease.jpg** - Screenshot of your FarmEase agriculture e-commerce platform
2. **solar-system.jpg** - Screenshot of your OpenGL solar system simulation
3. **wallet.jpg** - Screenshot of your digital wallet system interface
4. **tourmalawi.jpg** - Screenshot of your TourMalawi tourism website
5. **godot-vehicle.jpg** - Screenshot of your Godot vehicle simulation game

## How to Add Images:

1. Copy your project screenshots to this `public` folder
2. Rename them to match the names above (or update the paths in `src/components/Projects.tsx`)
3. Recommended image size: 800x600px or similar aspect ratio
4. Format: JPG or PNG

## If Images Are Missing:

The portfolio will automatically fall back to gradient backgrounds if images are not found, so your site will still look good!
